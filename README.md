# Space API

Space API's aim is to provide space data to developers in interesting format like JSON.


## Requirements

* Ruby
* Bundler gem


## Install

```console
$ git clone https://github.com/rhannequin/space-api-v2.git
$ cd space-api-v2
$ bundle install
```


## Launch

```console
$ bundle exec foreman start
```


## Test

```console
$ bundle exec rake specs
```
